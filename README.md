# Коммунальные счета (CommunalBills)

Настольное приложение (Windows, WPF) для удобного учёта коммунальных платежей: справочник поставщиков по вашему адресу и быстрый доступ к оплате через веб-банкинг во встроенном браузере.

## Возможности

- **🏢 Предприятия** — редактируемый список поставщиков коммунальных услуг: название, сайт и лицевой счёт по каждой услуге.
  - Выбор области подставляет типовых поставщиков газа/электроэнергии (черновик — поправьте под себя).
  - Лицевые счета можно вписать, изменить, скопировать в буфер одной кнопкой.
  - Все данные сохраняются локально (`%LOCALAPPDATA%\CommunalBills\settings.json`).
- **🔐 Privat24** — встроенный браузер открывает раздел коммунальных платежей Privat24; вход выполняется один раз, сессия сохраняется.
- **💳 Другой банк** — выбор банка (Ощадбанк, ПУМБ, Raiffeisen, Sense, monobank, Укрсиб) или агрегатора (iPay, EasyPay, Portmone) и открытие его веб-банкинга.
- **Проверка обновлений** — при запуске приложение проверяет наличие новой версии на GitHub и предлагает обновиться.

## Важно про данные

Приложение **не хранит и не передаёт** ваши пароли. Суммы счетов видны только после вашего входа в личный кабинет банка во встроенном браузере — это стандартная авторизация банка. Никакой сторонний сервис данные не получает.

## Установка

Скачайте установщик `CommunalBills-Setup-*.exe` из раздела [Releases](https://github.com/andrey1b/CommunalBills/releases) и запустите. Сборка self-contained — отдельная установка .NET не требуется. Нужен установленный **Microsoft Edge WebView2 Runtime** (обычно уже есть в Windows 10/11).

## Сборка из исходников

```bat
dotnet build -c Debug
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=false
```

## Технологии

- .NET 9 / WPF
- Microsoft Edge WebView2
- Inno Setup (установщик)
